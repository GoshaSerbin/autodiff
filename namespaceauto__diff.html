A shared ownership pointer to a Node.

Smart pointer is used to garantee that the node is alive if needed and can be deleted if not. Shared ownership is chosen because the same node can be required by many others. For example \(c = a * b\) and \(d = c * a\) means that \(a\) is required for both \(c\) and \(d\) nodes during the backpropagation.
