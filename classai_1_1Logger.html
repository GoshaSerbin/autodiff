Detailed Description

Static class for logging.

Based on the singleton pattern, thread safety. The default stream is std::cout. It uses std::vformat for formating. For custom classes logging you need to implement specialization of std::formatter for your class.

Setting output stream:

auto file = std::make_unique<std::ofstream>("log.log");
Logger::GetInstance().SetOutputStream(file.get());
